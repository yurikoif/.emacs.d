# Installation

Add this to `~/.config/alacritty/alacritty.yml`

## Light

```yaml
colors:
  primary:
    background: "0xfafafa"
    foreground: "0x4b505b"

  normal:
    black: "0x4b505b"
    red: "0xd15f5f"
    green: "0x608e32"
    yellow: "0xbe7e05"
    blue: "0x5079be"
    magenta: "0xb05ccc"
    cyan: "0x3a8b84"
    white: "0x949ba5"

  bright:
    black: "0x4b505b"
    red: "0xd15f5f"
    green: "0x608e32"
    yellow: "0xbe7e05"
    blue: "0x5079be"
    magenta: "0xb05ccc"
    cyan: "0x3a8b84"
    white: "0x949ba5"
```
